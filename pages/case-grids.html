Grid Systems

Grid Systems is a small grid-based layout system experiment with CSS Grid. It is a fully responsive website that displays the cover of the timeless classic book of Swiss designer Josef Müller-Brockmann.

Motivation & Goals

Before this project, when I was building a grid system I was using Bootstrap or some complicated custom HTML with cols, rows and containers done with flexbox… I learnt about this modern way to build layouts and decided to make this project so I could learn as much as possible about CSS Grid.


Grid Systems / Raster Systeme was my first graphic design book because I love neatly designed grids.

Tech Used

After exploring about how I could properly organize the CSS files, I learned about BEM and ITCSS architecture and decided to implement the last one. Since them, I use this kind of code organization in my projects.


The site had to be responsive too, so I created a SASS mixin with dynamic media queries. If I need to change the break points, I just have to do the change in the mixin file and it will be applied everywhere in the stylesheet.

Problems & Thought Process

As the grid was quite complicated, I first learnt how exactly CSS Grid works, which I did following the documentation and reading some blog posts.


Due to CSS Grid specifications, I had to create a custom solution to have the layout exact how I wanted to be. Once I got the grid solution, it was just a matter of implementing all the grid cells and the media queries.

Lessons Learned

I learnt a lot about how CSS Grid works and is implemented. I think that as a front-end developer, a basic but very important topic is to know how to create flexible and responsive grids.


After this project I am much more confident in how I create easy to use responsive layouts. Now I create all my projects with a similar approach, using CSS Grid for the layout and a SASS mixin for the media queries.
